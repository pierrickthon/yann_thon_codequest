### `sum(arr)`
Option boucle :
```javascript
function sum(arr) {
  let total = 0;
  for (let i = 0; i < arr.length; i++) {
    total += arr[i];
  }
  return total;
}
```

Option moderne :
```javascript
function sum(arr) {
  return arr.reduce((acc, val) => acc + val, 0);
}
```
